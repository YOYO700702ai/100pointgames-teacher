Firebase 後台資料庫設定教學

亞德雷大陸教師版 | 存檔資料庫設定說明

← 回到設定頁
💡 Firebase 是 Google 提供的免費雲端資料庫，用來儲存學生的遊戲進度與存檔資料。必須設定才能讓學生存檔。
1 建立 Firebase 專案
  • 前往 https://firebase.google.com
  • 點 「Get started in console」，用 Google 帳號登入
  • 點 「開始之前，請先設置 Firebase 專案」
  • 輸入專案名稱（例：我的班級遊戲）
  • 點 「繼續」，等幾秒後再點 「繼續」
  • 看到「您的 Firebase 專案已準備就緒」→ 點 「繼續」
2 建立 Firestore 資料庫
  • 進入專案後，左側選單點 「建構」→「Firestore Database」
  • 點 「建立資料庫」
  • 版本選 Standard 版 → 點 「下一步」
  • 位置選 asia-east1（距離台灣最近）→ 點 「下一步」
  • 安全性規則選 「以正式版模式啟動」 → 點 「建立」
3 設定讀寫權限
  • 進入 Firestore 後，點上方 「規則」 分頁
  • 找到 if false，把 false 改成 true
  • 點 「發布」
修改後的規則內容應如下：
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
4 取得設定碼
  • 點左上角 齒輪圖示 ⚙ → 「設定」
  • 點 「一般」
  • 下滑到 「你的應用程式」 區塊
  • 點 </> 網頁圖示
  • 應用程式暱稱隨便填（例：game）→ 點 「註冊應用程式」
  • 把出現的 整段程式碼 複製，貼到設定頁的欄位即可
⚠️ 注意事項：
• Firebase 免費方案每天有讀取上限（約 50,000 次），一般班級使用完全足夠
• 設定碼包含你的專案資訊，請勿分享給不相關的人
• 每位老師需要建立自己的 Firebase 專案，不能共用同一個

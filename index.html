填寫以下設定，即可獲得你班級專屬的遊戲網址與後台網址

① 班級資訊
② Google 試算表題庫網址

先複製這份範本試算表，換成你的題目，再點「檔案 → 共用 → 發布到網路 → CSV」複製網址貼到下方。

③ Firebase 設定（學生存檔用）
取得 Firebase 設定的步驟：
1. 前往 console.firebase.google.com → 建立專案
2. 左側選 Firestore Database → 建立資料庫（正式模式）
3. 點左上齒輪 ⚙️ → 專案設定 → 捲到最下方
4. 點 </> 網頁應用程式 → 取名後下一步
5. 複製出現的 firebaseConfig 整段貼到下方
✅ 設定完成！把以下兩個網址存起來：
🎮 遊戲網址（分享給學生）
🔧 後台管理網址（只給自己看）

⚠️ 請把這兩個網址存到記事本，重新設定後網址會改變。
